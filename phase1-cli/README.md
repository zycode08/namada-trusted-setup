## Contributor

To contribute to the ceremony simply give:

```shell
cargo run --bin phase1 --features=cli -- $(coordinator-ip:port)
```

You can also test a contribution against a local coordinator (running at `127.0.0.1:8000`) with:

```shell
make contribution
```
